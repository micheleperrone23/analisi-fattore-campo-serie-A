# Analisi Fattore Campo — Serie A ⚽

**Primo Progetto Data Science** | ITS ICT Piemonte

## Obiettivo
Rispondere alla domanda: *Esiste un vantaggio significativo per le squadre che giocano in casa in Serie A?*

## Dataset
Ultime 4 stagioni (2020/21 - 2023/24) da [football-data.co.uk](https://www.football-data.co.uk) — database open source con statistiche dettagliate.

## Funzionalità
Inserisci il nome di una squadra di Serie A e il programma analizza automaticamente:
- Media punti in casa vs trasferta
- Media punti contro le "big"
-  Distribuzione degli esiti (vittoria/pareggio/sconfitta)
- Grafici per ogni analisi
- Estrazione di un file CSV pulito e ottimizzato per l'analisi

## Stato del progetto
✅ Progetto completato — Data Collection, EDA, Analisi Statistica e Machine Learning

## Tecnologie utilizzate
Python | Jupyter Notebook | Pandas | Matplotlib
